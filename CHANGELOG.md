# Changelog — リタス（Litus）

東京理科大生向け非公式アプリ「リタス」の変更履歴。
事前登録はこちら → https://lms.waiteu.dev/app

## v1.0.0（2026-07-07 ビルド・初版公開に向けた実験配布）

初回リリース。LETUS課題の自動収集・締切通知、CLASS時間割、モバイル出席登録のワンタップ化をスマホ単体で提供する。

### 追加

- **初回チュートリアル**: 初回起動でアプリの使い方（課題見張り・出席ワンタップ・時間割と通知）を3枚のスライドで紹介し、そのままTUSアカウントでログイン（認証情報は保存しません）
- **起動画面**: 2回目以降の起動はリタスの起動画面でCLASS/LETUSのログイン状態を自動確認してから入場。以後の出席・課題・通知がスムーズに
- **課題の自動更新**: アプリ起動後、バックグラウンドでLETUSの課題を自動収集し、一覧と通知を最新化
- **出席ページへの自動到達**: 出席タブを開くとCLASSの入口→ポータル→モバイル出席登録まで自動で遷移。認証コードを入れて押すだけ
- **通知チャンネル**: Androidの通知チャンネル「出席リマインド」（最優先・ヘッドアップ表示）と「課題リマインド」を分離。アプリを開いている間も通知が表示されるように

### 安定性

- CLASSの「システムエラー」（セッション競合）を検知して自動復帰
- 長時間放置→復帰時にログイン状態とページを自動で立て直し、通知予約も貼り直し
- 収集がページ読込で詰まった場合の15秒タイムアウト保険

### 開発メモ

- Chrome拡張「LETUS課題見張り番」（lms-task-watcher）とバックエンドを共用。ロードマップ: フェーズ2=CLASS連携 → フェーズ3=リタス連携
- バージョン系列は本リポジトリ独自に v1.0.0 から開始（旧称 v2.0.0 は使用しない）
